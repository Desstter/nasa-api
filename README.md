# NASA API Explorer

Explorador de la [NASA Open API](https://api.nasa.gov/). Visualiza la imagen astronómica del día (APOD), datos de asteroides cercanos, imágenes de Marte y más datos del espacio en tiempo real.

## Stack

JavaScript · HTML · CSS

## Features

- **APOD** — Astronomy Picture of the Day con descripción
- **Asteroides** — Near Earth Objects con datos orbitales
- **Mars Rover** — fotos de los rovers en Marte
- Interfaz inmersiva de temática espacial

## Setup

1. Obtén tu API key gratuita en [api.nasa.gov](https://api.nasa.gov/)
2. Reemplaza `DEMO_KEY` en `script.js` con tu key
3. Abre `index.html` en el navegador

```bash
npx serve .    # Opcional: servidor estático local
```
